# CodeAlpha_StudentGradeTracker_
# 📌 Overview
This is a simple Java program that takes input of student marks and calculates:
- Average marks
- Highest marks
- Lowest marks

## 🚀 Features
- User input using Scanner
- Dynamic storage using ArrayList
- Calculation of statistics (average, max, min)
- Clean and structured output

## 🛠️ Technologies Used
- Java
- ArrayList
- Scanner class

## 📷 Sample Output
Enter number of students: 5
Enter marks of student 1: 34  
Enter marks of student 2: 56  
Enter marks of student 3: 98
Enter marks of student 3: 44
Enter marks of student 3: 12


----- Summary Report -----  
Average Marks: 48.8
Highest Marks: 98  
Lowest Marks: 12  

## 📚 Concepts Used
- Loops (for loop)
- Conditional statements (if-else)
- Functions (methods)
- Object-Oriented Programming basics
